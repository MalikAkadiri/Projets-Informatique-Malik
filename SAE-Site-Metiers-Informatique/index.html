Les métiers de l'informatique

Dans ce petit site, nous allons vous présenter plusieurs débouché que vous pouvez avoir à la sortie d'un BUT Informatique. Nous avons divisé le site en trois parties, une partie sur le côté créatif, une autre sur développement d'applications, et une sur le Web. Dans chacune de ces parties, vous pourrez retrouver une description détaillée des métiers ainsi que les salaires que l'on peut espérer.
